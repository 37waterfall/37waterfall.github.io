快速掌握 AJAX - 基础

AJAX（Asynchronous JavaScript and XML）

可以通过 JavaScript 直接获取服务端最新的内容而不必重新加载页面。让 Web 更能接近桌面应用的用户体验。

AJAX 就是浏览器提供的一套 API，可以通过 JavaScript 调用，从而实现通过代码控制请求与响应。实现通过 JavaScript 进行网络编程。

AJAX总结为以下四类：

  1. 按需获取数据
  2. 对用户输入的数据进行校验
  3. 自动更新页面上内容
  4. 提升用户体验 - 无刷新的体验

XMLHttpRequest 类型
readystatechange 事件是在 xhr 对象状态变化时触发（不单是在得到响应时）
本质上 XMLHttpRequest 就是 JavaScript 在 Web 平台中发送 HTTP 请求的手段

通常在一次 GET 请求过程中，参数传递都是通过 URL 地址中的 ? 参数传递。
xhr.open('GET', '/delete?id=1')

POST 请求过程中，都是采用请求体承载需要提交的数据。

xhr.open() 方法第三个参数要求传入的是一个 bool 值，
其作用就是设置此次请求是否采用异步方式执行，默认为 true，如果需要同步执行可以通过传递 false 实现：

缓存问题

这个办法的核心就是让浏览器认为每次请求的地址都是不同的。
xhr.open(‘GET’, ‘/time?t=’ + Date.now())
