Basic

Layout布局

  1. 基础布局

    通过row和col组件，，这是层级关系
    并通过col组件的span属性，这里指的是跨度
    element-ui每一行共分为24的跨度

  2. 分栏间隔

    指的是每个col组件之间的距离
    用gutter属性来定义，默认0
    跨度依旧是24

  3. 混合布局

    跨度分为12/6/6
    8/4/4/8等

  4. 分栏偏移

    指的是空出一段距离，
    按span的跨度来空距离

  5. 对齐方式

    结合flex布局
    给row即父元素设置
    <el-row type="flex" class="row-bg" justify="center">

  6. 响应式布局

    参照Bootstrap，设计了五个相应尺寸
    xs,sm,md,lg,xld

Container布局容器

布局容器组件，快速搭建页面基本结构
el-container，为最外层的结构，包裹其他容器
不同的布局，指的的结构不同
用的是flex布局

Icon图标

直接通过设置类名为 el-icon-iconName 来使用即可
<i class="el-icon-edit"></i>

Button按钮

使用type、plain、round和circle属性来定义 Button 的样式。

  • 可设置禁用状态
    文字按钮
    图标按钮
    加载中
    不同尺寸
    按钮组
    使用<el-button-group>标签来嵌套你的按钮。

Link文字链接

设置超链接的文字
可设置禁用状态
下划线
图标

Select选择器

v-model的值为当前被选中的el-option的 value 属性值
主要通过v-for来实现，渲染已创建好的数据[{},{}]

Notice

Alert警告

用于页面中展示重要的提示信息。
页面中的非浮层元素，不会自动消失。–干嘛的？？？

  • Alert 组件提供了两个不同的主题：light和dark。
    自定义关闭按钮
    带有 icon
    文字居中
    带有辅助性文字介绍

Message消息提示

常用于主动操作后的反馈提示。与 Notification 的区别是后者更多用于系统级通知的被动提醒。

  • 从顶部出现，3 秒后自动消失。
    用来显示「成功、警告、消息、错误」类的操作反馈。
    可以添加关闭按钮。
    使用 center 属性让文字水平居中。
    message 属性支持传入 HTML 片段
    可按需引入

  • 模拟系统的消息提示框而实现的一套模态对话框组件，
    用于消息提示、确认消息和提交内容。

Notification通知

悬浮出现在页面角落，显示全局的通知提醒消息。

  • 带有 icon

    让 Notification 偏移一些位置

Navigation

NavMenu导航菜单

重点！！！
为网站提供导航功能的菜单。

  • 顶栏
    侧栏-垂直菜单，可内嵌子菜单。
    折叠-展开收起

Tabs标签页

点击切换
分隔内容上有关联但属于不同类别的数据集合。

  • 选项卡样式
    卡片化

    自定义标签页
    动态增减标签页

Breadcrumb面包屑

  • 显示路径
    显示当前页面的路径，快速返回之前的任意页面。

PageHeader页头

一个返回，一个标题
如果页面的路径比较简单，推荐使用页头组件而非面包屑组件。

Dropdown下拉菜单

将动作或菜单折叠到下拉菜单中。

  • 移动到下拉菜单上，展开更多操作。
    可使用按钮触发下拉菜单
    可以配置 click 激活或者 hover 激活。
    指令事件-点击菜单触发事件
    不同尺寸

Steps步骤条

点击下一步。。。
引导用户按照流程完成任务的分步导航条，可根据实际应用场景设定步骤，步骤不得少于 2 步。

  • 含状态步骤条
    有描述的步骤条
    居中的步骤条-样式
    带图标的步骤条
    竖式步骤条
    简洁风格的步骤条

Others

Dialog对话框

  • Dialog 弹出一个对话框，适合需要定制性更大的场景。
    Dialog 组件的内容可以是任意的，甚至可以是表格或表单
    如果需要在一个 Dialog 内部嵌套另一个 Dialog

Tooltip文字提示

常用于展示鼠标 hover 时的提示信息。

  • 方向很多
    Tooltip 组件提供了两个不同的主题：dark和light。

Popover弹出框

Popover 的属性与 Tooltip 很类似

Card卡片

实用-展示信息
将信息聚合在卡片容器中展示。

  • 包含标题，内容和操作。
    卡片可以只有内容区域。
    带图片
    卡片阴影

Carousel走马灯

轮播图
在有限空间内，循环播放同一类型的图片、文字等内容

  • 指示器
    切换箭头
    卡片化
    方向-滑动方向

Collapse折叠面板

  • 通过折叠面板收纳内容区域
    手风琴效果-每次只能展开一个面板
    自定义面板标题

Timeline时间线

实用-博客里用到了
可视化地呈现时间流信息。

Timeline 可拆分成多个按照时间戳正序或倒序排列的 activity，时间戳是其区分于其他控件的重要特征，使⽤时注意与 Steps 步骤条等区分。

  • ⾃定义节点样式
    ⾃定义时间戳

Divider分割线

区隔内容的分割线。

垂直分割
<el-divider direction="vertical"></el-divider>

Calendar日历

  • 显示日期
    自定义内容
    自定义范围

Image图片

图片容器，在保留原生img的特性下，支持懒加载，自定义占位、加载失败等

  • 占位内容
    加载失败
    懒加载
    大图预览-点击大图

Backtop回到顶部

返回页面顶部的操作按钮

Drawer抽屉

类似Dialog，
Drawer 拥有和 Dialog 几乎相同的 API, 在 UI 上带来不一样的体验.

  • 呼出一个临时的侧边栏, 可以从多个方向呼出
    不添加 Title
