Basic

Layout布局

  1. 基础布局

    通过row和col组件，，这是层级关系
    并通过col组件的span属性，这里指的是跨度
    element-ui每一行共分为24的跨度

  2. 分栏间隔

    指的是每个col组件之间的距离
    用gutter属性来定义，默认0
    跨度依旧是24

  3. 混合布局

    跨度分为12/6/6
    8/4/4/8等

  4. 分栏偏移

    指的是空出一段距离，
    按span的跨度来空距离

  5. 对齐方式

    结合flex布局
    给row即父元素设置
    <el-row type="flex" class="row-bg" justify="center">

  6. 响应式布局
    参照Bootstrap，设计了五个相应尺寸
    xs,sm,md,lg,xld
